# Absensi-Sekolah-2

Sistem Absensi Siswa Berbasis QR Code

## Tentang Proyek

Absensi-Sekolah-2 adalah aplikasi absensi siswa berbasis QR Code yang dikembangkan menggunakan Laravel.

Proyek ini dibuat untuk mendigitalisasi proses absensi siswa di sekolah sehingga lebih cepat, akurat, dan mudah dipantau.

Target implementasi:

* SMP Negeri 9 Denpasar
* Absensi siswa menggunakan kartu QR
* Monitoring kehadiran siswa
* Monitoring BK
* Rekap dan laporan absensi
